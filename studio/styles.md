# 公众号排版风格预设（V6 · 12 种）

## 通用排版基础

正文 15px，行高 2.0，字间距 1px，段间距 20px，无首行缩进，内容区最大宽度 578px。

## 设计铁律

1. **不做按钮式标题** — 全宽铺开，线条和字重建立层级
2. **全部浅色底** — 不用深色背景（避免微信深色模式反转）
3. **纯内联样式** — 不用 class 不用 `<style>` 块
4. **图片框紧贴图片** — padding:0 + font-size:0/line-height:0 消除 inline 间隙
5. **禁用** — position / box-shadow / text-shadow / linear-gradient / @font-face / animation

## 5 种图片角色

| 角色 | Markdown 语法 | 视觉特征 |
|------|--------------|---------|
| 配图 | `![[1]]` | 风格化边框 + 图注 |
| 封面 | `![[1\|cover]]` | 全宽大图，无边框，COVER 标签 |
| 引用 | `![[1\|quote]]` | 左侧粗色条 + 浅底 + QUOTE 标签 + 内层白底 |
| 双图 | `![[1,2]]` | 两张并排，间隔由风格决定 |
| 三图 | `![[1,2,3]]` | 三张并排 |

## 新增版式元素

- **导语** `>>> 内容` — 文章开头的概括段
- **提示框** `!!! tip/note/warn/info 标题\n内容\n!!!` — 4 种类型
- **金句** `‖ 内容 ‖` — Pull Quote 大字突出
- **表格** 标准 Markdown 表格语法
- **落款** `--- 作者：xxx / 公众号：xxx`
